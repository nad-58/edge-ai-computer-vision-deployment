# Production Readiness Checklist

Before release, an edge AI model should be reviewed for both model quality and deployment engineering readiness.

## Readiness checklist

- Intended use and operating environment defined
- Representative validation data available
- Target hardware and runtime defined
- Accuracy acceptance criteria met
- Latency acceptance criteria met
- Memory budget met
- Quantisation impact assessed
- Robustness testing completed
- Input-quality handling defined
- Integration tests completed
- Release documentation prepared
- Monitoring or field-feedback process defined
- Update and rollback process defined

## Release decision

A model should not be considered ready only because accuracy is acceptable. Runtime performance, integration, robustness, and maintainability must also be demonstrated.
